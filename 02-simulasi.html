Cara Penyelesaian

Translasi

Geser titik $(x,y)$ sejauh $(a,b)$ tanpa mengubah bentuk.

\[ T(x,y) = (x+a,\; y+b) \]
Langkah Penyelesaian
  1. Tentukan titik asal $(x, y)$ dan vektor translasi $(a, b)$.
  2. Tambahkan komponen: $x' = x + a$
  3. Tambahkan komponen: $y' = y + b$
  4. Bayangan: $(x', y')$
Penyelesaian Titik Anda
Buat titik di canvas lalu klik TERAPKAN untuk melihat langkah penyelesaiannya.
Refleksi (Pencerminan)

Cerminkan titik terhadap sumbu, garis, atau titik pusat tertentu.

Sumbu-X
\[ R_x(x,y) = (x,\; -y) \]
Sumbu-Y
\[ R_y(x,y) = (-x,\; y) \]
Garis $x = k$
\[ R_{x=k}(x,y) = (2k-x,\; y) \]
Garis $y = k$
\[ R_{y=k}(x,y) = (x,\; 2k-y) \]
Garis $y = x$
\[ R_{y=x}(x,y) = (y,\; x) \]
Garis $y = -x$
\[ R_{y=-x}(x,y) = (-y,\; -x) \]
Titik Pusat $O(0,0)$
\[ R_O(x,y) = (-x,\; -y) \]
Langkah Penyelesaian
  1. Tentukan titik asal $(x, y)$ dan jenis cermin.
  2. Terapkan rumus yang sesuai.
  3. Bayangan: $(x', y')$
Penyelesaian Titik Anda
Buat titik di canvas lalu klik TERAPKAN untuk melihat langkah penyelesaiannya.
Rotasi (Perputaran)

Putar titik $(x,y)$ sebesar sudut $\theta$ terhadap pusat $O(0,0)$.

Rotasi $90°$ (berlawanan jarum jam)
\[ R_{90°}(x,y) = (-y,\; x) \]
Rotasi $-90°$ (searah jarum jam)
\[ R_{-90°}(x,y) = (y,\; -x) \]
Rotasi $180°$
\[ R_{180°}(x,y) = (-x,\; -y) \]
Rotasi $270°$ (berlawanan jarum jam)
\[ R_{270°}(x,y) = (y,\; -x) \]
Rotasi $-270°$ (berlawanan jarum jam)
\[ R_{-270°}(x,y) = (-y,\; x) \]
Rumus Umum (sudut $\theta$)
\[ R_\theta(x,y) = (x\cos\theta - y\sin\theta,\; x\sin\theta + y\cos\theta) \]
Langkah Penyelesaian
  1. Tentukan titik $(x, y)$, besar sudut, dan arah rotasi.
  2. Pilih rumus sesuai sudut rotasi.
  3. Substitusi nilai $x$ dan $y$.
  4. Bayangan: $(x', y')$
Penyelesaian Titik Anda
Buat titik di canvas lalu klik TERAPKAN untuk melihat langkah penyelesaiannya.
Dilatasi (Perkalian)

Perbesar atau perkecil titik $(x,y)$ dengan faktor skala $k$ terhadap pusat tertentu.

Terhadap Pusat $O(0,0)$
\[ D_k(x,y) = (kx,\; ky) \]
Terhadap Pusat $P(a,b)$
\[ x' = a + k(x - a) \] \[ y' = b + k(y - b) \]
Langkah Penyelesaian
  1. Tentukan titik $(x, y)$, faktor skala $k$, dan pusat dilatasi.
  2. Kalikan koordinat dengan $k$ (jika pusat $O(0,0)$).
  3. Atau gunakan rumus pusat $P(a,b)$ untuk pusat lainnya.
  4. Bayangan: $(x', y')$
Penyelesaian Titik Anda
Buat titik di canvas lalu klik TERAPKAN untuk melihat langkah penyelesaiannya.
